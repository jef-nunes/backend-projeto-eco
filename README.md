## Resumo
Protótipo de backend (REST API) para um site com a temática "desastres naturais e meio ambiente". A aplicação foi desenvolvida com a linguagem Java e o framework Spring Boot. Entre os recursos implementados estão:
- Integração a banco de dados MySQL
- Operações CRUD das principais entidades
- Operações de adicionar e remover relacionamentos entre entidades
- Gerenciador de exceções global
- Resposta JSON padronizada
